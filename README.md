# 🚀 NexusAssistant

NexusAssistant is a desktop productivity assistant developed in **C++** using **Visual Studio** and the **ICB GUI Framework**.

The project combines multiple daily productivity tools into a single desktop application, providing users with an organized and user-friendly interface for managing everyday activities.

---

## ✨ Features

- 📅 Schedule & Task Management
- 🔔 Notifications & Reminders
- 🎉 Special Days Tracker
- 🌤️ Weather Information
- ⚽ Sports Updates
- 🚗 Vehicle Service Tracking
- 💰 Personal Finance Tracking
- 📈 Portfolio Monitoring
- 💪 Workout Progress Tracking
- 🤖 AI Assistant Module
- ⚙️ User Settings

---

## 🛠️ Technologies

- C++
- Visual Studio
- Win32 API
- GDI+
- WinHTTP
- ICB GUI Framework (Custom GUI Library)

---

## 🖥️ About the GUI Framework

This project was built using the **ICB GUI Framework**, a custom C++ graphical user interface library developed for educational purposes.

The framework provides reusable UI components, image rendering, event handling, and window management on top of the Win32 API, allowing developers to build desktop applications more efficiently while focusing on application logic.

## 🎯 Project Purpose

The purpose of this project is to demonstrate desktop software development using modern C++ concepts together with a custom GUI framework.

It integrates multiple independent utility modules into a single desktop application while emphasizing clean interface design and modular software architecture.
